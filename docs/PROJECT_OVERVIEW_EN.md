# Retail Analytics Dashboard

## Project Overview

Retail Analytics Dashboard is an end-to-end business intelligence project demonstrating data analytics skills from data generation and storage to reporting and visualization in Power BI.

The project is based on a synthetic dataset representing a retail chain selling home and household products.

---

# Project Goals

* Build an analytical data warehouse
* Implement a Star Schema model
* Automate data generation
* Load data into PostgreSQL
* Create interactive Power BI dashboards
* Demonstrate Data Analytics and Business Intelligence skills
